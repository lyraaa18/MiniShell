# 🐚 Mini Shell v2

Mini Shell v2 adalah emulator shell sederhana berbasis Python yang memungkinkan pengguna untuk mengeksekusi perintah dasar sistem melalui antarmuka CLI (Command Line Interface). 
Aplikasi ini dapat digunakan untuk navigasi file dan direktori, membuat dan menghapus file, serta menampilkan isi direktori.

## 📦 Fitur

- Menampilkan daftar file dan folder dalam direktori saat ini
- Berpindah antar direktori (`cd`)
- Membuat file (`touch`)
- Menghapus file (`rm`)
- Membuat folder (`mkdir`)
- Menghapus folder (`rmdir`)
- Menampilkan isi file (`cat`)
- Membantu pengguna dengan perintah `help`
- Keluar dari shell menggunakan `exit`
